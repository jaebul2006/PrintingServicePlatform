소프트웨어

3DP 소프트웨어는 3D Printing에 필요한 소프트웨어를 다운 받을 수 있는 서비스 입니다.

소프트웨어

  • MediACE3D - AFO

    MEDIACE3D-AFO는 하지보조기전용 3D설계도구로 3D 프린터에 특화된 환자맞춤형 하지보조기 제작을 지원하는 의료 3D CAD 소프트웨어입니다.
