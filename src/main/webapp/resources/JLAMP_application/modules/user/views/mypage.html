비밀번호 변경

- 비밀번호를 변경하시면 기존 로그인 계정은 로그아웃 됩니다.

- 쉬운 비밀번호나 자주 쓰는 사이트의 비밀번호가 같을 경우, 도용 되기 쉬우므로 주기적으로 변경하셔서 사용하는 것이 좋습니다.

- 아이디와 주민등록번호, 생일, 전화번호 등 개인정보와 관련된 숫자, 연속된 숫자, 반복된 문자 등 다른 사람이 쉽게 알아 낼 수 있는 비밀번호는 사용을 자제해 주시기 바랍니다.

- 비밀번호는 최소 8~15자리까지 영문자, 특수문자, 숫자를 조합하여 사용 가능합니다.
